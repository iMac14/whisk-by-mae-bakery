# Whisk by Mae Bakery

Welcome to Whisk by Mae Bakery! This is a web application for managing a bakery's online presence, including customer orders, admin dashboard, and product management.

## Features

- Customer registration and login
- Product browsing and ordering
- Shopping cart functionality
- Admin panel for managing products, orders, and customers
- Secure authentication system

## Setup

1. Clone the repository
2. Set up your database using the provided schema.sql
3. Configure your database connection in db.php
4. Run the application on a PHP server (e.g., XAMPP)

## Technologies Used

- PHP
- MySQL
- HTML/CSS/JavaScript
- Bootstrap (for styling)
